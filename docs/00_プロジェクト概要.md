# プロジェクト概要

## 1. 目的
本プロジェクトは、社内エンジニア向けの技術研修で利用することを目的としたサンプルアプリケーションです。

## 2. 研修で得られる効果
- Java と Spring Boot を用いたWebアプリケーション開発の基礎を理解する。
- Spring Data JPA を利用したデータベース操作（CRUD）を習得する。
- データベース設計（テーブル定義）とアプリケーション実装の関連性を学ぶ。
- GitHub Codespaces を活用したモダンなクラウド開発環境に慣れる。
- 業務アプリケーション開発の基本的なプロセスと、ものづくりの楽しさを体験する。

## 3. 技術スタック
- **言語**: Java 21
- **フレームワーク**: Spring Boot 3.3.0
- **テンプレートエンジン**: Thymeleaf
- **データベースアクセス**: Spring Data JPA (Hibernate)
- **開発用データベース**: H2 Database
- **ビルドツール**: Maven
- **その他**:
  - Lombok: コードの記述を簡略化
  - springdoc-openapi: API仕様書の自動生成
  - Spring Boot Actuator: アプリケーションの監視

## 4. アプリケーションの起動方法
1. GitHub Codespaces でこのリポジトリを開きます。
2. ターミナルで以下のコマンドを実行するか、VS Codeの `Run and Debug` パネルから `Spring Boot` アプリケーションを起動します。
   ```bash
   ./mvnw spring-boot:run
   ```
3. 起動後、VS Codeがポートフォワーディングの通知を出すので、ブラウザで開きます。
4. `http://localhost:8082/assets` にアクセスすると、資産一覧画面が表示されます。
   - H2データベースのコンソールは `http://localhost:8082/h2-console` からアクセスできます。
