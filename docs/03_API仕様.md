# 画面・機能仕様

本アプリケーションが提供する画面と、それに関連する機能（HTTPリクエスト）の一覧です。

---

## 資産管理

### 資産一覧画面
- **URL**: `GET /assets`
- **機能**: 登録されている資産を一覧で表示する。

### 資産登録画面
- **URL**: `GET /assets/new`
- **機能**: 新しい資産を登録するためのフォーム画面を表示する。

### 資産登録処理
- **URL**: `POST /assets/new`
- **機能**: 新しい資産をデータベースに登録する。

| パラメータ | 型 | 説明 |
|---|---|---|
| name | String | 資産名 |
| location | String | 保管場所 |

---

## 貸出管理

### 貸出管理画面
- **URL**: `GET /loans`
- **機能**: 現在の貸出一覧、貸出可能な資産、ユーザの一覧を表示する。

### 貸出入力画面
- **URL**: `GET /loans/new`
- **機能**: 特定の資産を貸し出すための入力画面を表示する。

| パラメータ | 型 | 説明 |
|---|---|---|
| assetId | Long | 貸し出す資産のID |

### 貸出処理
- **URL**: `POST /loans`
- **機能**: 資産の貸出処理を行う。

| パラメータ | 型 | 説明 |
|---|---|---|
| assetId | Long | 貸し出す資産のID |
| userId | Long | 借りるユーザのID |
| loanDate | String | 貸出日 (yyyy-MM-dd) |
| periodDays | Integer | 貸出期間（日数） |

### 返却処理
- **URL**: `POST /loans/return`
- **機能**: 貸出中の資産を返却する。

| パラメータ | 型 | 説明 |
|---|---|---|
| loanId | Long | 返却対象の貸出ID |
